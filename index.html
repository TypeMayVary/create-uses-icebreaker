Unlikely uses!

This is a (hopefully) fun exercise to help you break the ice and engage in some creative thinking.

  1. Each table gets an object from the list below.
  2. Between you, think of wild, fun, or clever uses for it. This must be relevant to one of the teams in the department.
  3. Decide on the table's favourite idea. The more creative and out-of-the-box your idea is, the better!
  4. We'll finish off with sharing each table's favourite idea with the rest of the group.

Example Creative Uses

  • An empty pizza box: Used as the official inbox for all communications into the department.
  • A tangled ball of charging cables: Used as a stress-relief fidget toy for the whole office.
  • A cracked monitor: Turned into a window to the "multiverse" for X team's daily standup.

Objects

  • 1. One extra long line of code
  • 2. A wireless mouse that doesn't work
  • 3. A post-it note with a password written on it
  • 4. A stack of Jira tickets from 2019
  • 5. An unlabelled USB stick found in the car park
  • 6. A Google Chrome browser with 65 open tabs
  • 7. One noisy mechanical keyboard
  • 8. An Excel spreadsheet called finance_v1.5_final_FINAL.xlsx
  • 9. A .txt file with all the latest buzzwords
